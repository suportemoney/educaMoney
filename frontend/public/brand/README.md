# Brand assets — EducaMoney

Coloque aqui os arquivos oficiais da marca. O site já aponta para estes caminhos.

## Logo (proporção 1:5)

| Item | Valor |
|------|--------|
| Arquivo | `logo.png` (preferencial) ou `logo.svg` (placeholder atual) |
| Proporção | **1:5** → largura : altura (ex.: **1000×200** px) |
| Fundo | **Transparente** (PNG/SVG) |
| Uso | Header e hero da landing |

Como exportar:

1. Exporte o logo horizontal em PNG com canal alpha.
2. Mantenha a proporção 5 de largura para 1 de altura.
3. Substitua `logo.png` nesta pasta (mesmo nome).
4. Sem alterar código: recarregue o site.

O HTML tenta `logo.png` primeiro; se não existir, usa `logo.svg`.

## Favicon

| Item | Valor |
|------|--------|
| Arquivo | `favicon.png` (preferencial, 32×32 ou 48×48) ou `favicon.svg` |
| Fundo | Transparente ou cor sólida da marca |
| Uso | Ícone da aba do navegador (`index.html`) |

Substitua `favicon.png` nesta pasta e, se quiser, mantenha o SVG como fallback.
